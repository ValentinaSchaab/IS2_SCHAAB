# Ingeniería de Software II - Valentina Schaab

## Descripción
Este repositorio corresponde a la materia Ingeniería de Software II - Dr. Pedro Colla.

### Objetivo del proyecto
El objetivo es visualizar y gestionar los proyectos y trabajos prácticos utilizando herramientas como Git, GitHub y Python.

---

## Estructura del repositorio

### Carpetas principales
- src: código fuente de los programas  
- doc: documentación del proyecto  
- bin: archivos ejecutables o resultados  
- script: scripts auxiliares  

### Contenido actual
1. Programa en Python para calcular números primos  
2. Uso de Git para control de versiones  
3. Sincronización con repositorio remoto  

---

## Uso del proyecto

### Pasos para ejecutar el programa
1. Clonar el repositorio  
2. Acceder a la carpeta `src`  
3. Ejecutar el archivo `primos.py`
